# Contributing to COD:MW4

Thank you for your interest in contributing to COD:MW4.

## Important Notice

COD:MW4 is a closed source project. The source code is not publicly available. Only approved contributors have access to the codebase.

## How to Apply

1. Join the Discord server: https://discord.gg/ZaGdCPGWzd
2. Post in the recruitment channel with your information
3. Wait for review from El

## For Programmers

If accepted as a programmer:

- You will sign a confidentiality agreement
- You will receive access to the private code repository
- You must follow the coding standards documented in the specifications
- You may NOT share any code with anyone outside the programmer team
- Violation results in immediate removal and legal action

## For Testers

If accepted as a tester:

- You will receive access to the test server
- You must follow the bug reporting format
- Report bugs honestly without exploiting them
- Do not share test server access with others

## Code of Conduct

All contributors must follow our Code of Conduct (CODE_OF_CONDUCT.md).

## Questions

Join Discord and ask in the appropriate channel.
